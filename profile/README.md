# How do I join?

Please [schedule a chat with Dawn][schedule].

# Why join?

Because you'll learn and level-up on numerous skills:

- Communication
- Collaboration
- Rust
- Knowledge sharing
- Various development tools and workflows

You'll make friends.

You'll have fun.

You may build something you'll be proud of.
